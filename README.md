<h1 align="center">puck</h1>  
<p align="center">  
 <b> Powerful media server backend application, written in Rust, for downloading and reading manga.</b>  
</p> 

Setup Project on Clone

```
cd src
go build setup.go
```

<p><b>Build Project</b></p>
Prerequisites: Cargo, Yarn

```
go build build.go
```
